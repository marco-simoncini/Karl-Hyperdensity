# Hyperdensity Control Room UI Concept

Hyperdensity UI direction is not a generic Kubernetes dashboard.

It is the **Control Room del Grande Padre** for shell-runtime governance.

## Primary Surfaces

- Control Room
- Shell Compliance Matrix
- Resource Equilibrium
- Fleet Onboarding
- Recommendation/Action Feed
- Risk & Safety Center
- Overcommit/Economics View

## Permanent Data Axes

Every major view always shows:
- Declared
- Runtime
- Observed

## Core Vocabulary

The UI language is explicitly product vocabulary:
- Shell
- Runtime Overlay
- Compliance
- Equilibrium
- Donor
- Receiver
- Burst
- Reclaim
- Evidence
- Risk
- Guard
- Rollback
- Authority
