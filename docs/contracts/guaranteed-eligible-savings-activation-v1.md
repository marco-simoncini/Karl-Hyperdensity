# KARL Hyperdensity — Guaranteed Eligible Savings Activation v1

**Contract ID:** `hyperdensity_guaranteed_eligible_savings_activation_v1`  
**Milestone:** `hyperdensity_guaranteed_eligible_savings_activation_v1`

## Product definition

> KARL Hyperdensity can activate guaranteed eligible savings for verified resource movements that pass accounting, SLO, rollback, unit-price, audit and exclusion gates, while excluding estimated, synthetic, evidence-gated and non-verified value.

## Sprint 11A scope

- `guaranteedEligibleSavingsAllowed=true` for **scoped eligible records only**
- `guaranteedSavingsScope=scoped_eligible_records`
- Universal guarantee remains **disabled**

## Forbidden

- universal guaranteed savings / all workloads
- estimated, synthetic, reference, Windows evidence-gated, placeholder price as guaranteed
- Dashboard/FluidVirt/Inventory as accounting or guarantee authority
